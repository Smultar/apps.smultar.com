Version 3.0.1

5/23/2019

-Fixed an issue using the mute command in role mode, prometheus didn't alert the target user of being muted.
-Fixed an issue when someone left a discord server, prometheus would sometimes report an incorrect stay duration.
-Prometheus is scared of ghost, this explains why he crashes to phantom images.
-Fixed an automatic unmute issue, were the database would crash if no muted users were detected.
-Fixed a critical crash that happened when ever someone asked prometheus if she was a cat girl...
-Fixed an issue were prometheus didn't created a server owner registry to a server in his database, rendering the server ownerless.
-Fixed an edit message history issue, were prometheus would report "." for embed messages popping up in a server.
-Fixed an deleted message history issue, were prometheus would report "." for deleted embed messages popping up in a server.
-Fixed a bot on bot error, that would cause prometheus to crash when another bot uploaded an embed message with no image, making the user's displayName appear as null.
-Corrected prometheus's warning message to other users, that displayed a random *.
-Corrected the infraction add/remove command, to display target user's information and not the command author.
-Fixed a crash, when prometheus try's to apply infraction to a user that doesn't exist in the current server.
-Prometheus forgot to display warning messages when ever someone uses the /lookup command anywhere.
-Fixed an issue where the lookup command didn't display the correct information sometimes.
